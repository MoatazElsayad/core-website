# Munich Bridge

A single-page React + Tailwind consultant-style website for the course
`CORE: Freshmen Seminar - Imagining Exile`.

This project uses realistic placeholder content for a relocation and integration
hub supporting Syrian families preparing for life in Munich. Replace the dummy
content in `src/data/siteContent.js` with verified research later.

## Scripts

```bash
npm install
npm run dev
npm run build
```
